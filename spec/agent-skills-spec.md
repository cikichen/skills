# Agent Skills Specification

This repository follows a `skills/<skill-name>/` layout.

## Required files per skill

- `SKILL.md`
- `README.md`
- `LICENSE`

## Recommended files

- `README_CN.md`
- `references/`
- `CHANGELOG.md`
- `CONTRIBUTING.md`

## Naming

- Skill directory name should be kebab-case.
- Keep scope narrow: one skill directory should represent one skill.
